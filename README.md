Things I did not know before starting this onboarding program.

## Render tree
Elements with `display: none` are not included in render tree as they do not appear in the rendered output.
But the ones with `visibility: hidden` are included as they do take up space.

2. Largest Contentful Paint
3. First Contentful Paint
4. Time To Interactive
